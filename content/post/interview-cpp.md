# 基础

## const

### 函数const

1. 如果函数需要传入一个指针，是否需要为该指针加上const，把const加在指针不同的位置有什么区别；

   ``` cpp
   void fun1(const int * p); //这个可以禁止函数通过传入指针改变对应的变量。
   void fun2(int * const p); //这个传入常指针，也就是禁止这个指针指向别的地方，因为这是传值的，所以这个const是无意义的。
   ```

2. 如果写的函数需要传入的参数是一个复杂类型的实例，传入值参数或者引用参数有什么区别，什么时候需要为传入的引用参数加上const。 

   传入值参数的时候，会有构造和析构的过程，会消耗时间，如果是引用参数，那么将会提高效率。对于非内部数据类型的输入参数都应该通过常引用传递，对于内置数据类型的输入参数应该通过值传递，否则无法提高效率之外还降低可理解性。
